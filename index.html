Wikipedia Map

A tool for exploring connections between topics.

How to use

  • Top Bar: Type topics (e.g., "Cats", "Physics") and press Go. Use Random for a surprise.
  • Controls (Bottom Right):
    • Find: Zoom to and select a random node.
    • Expand: Show connections for the selected node.
    • Wiki: Open the article in a new tab.
    • Delete: Remove the selected node.
  • People Mode: Toggle in top bar to find only people/characters.
